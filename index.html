My Projects

The software application that I am most proud of is the NiteTrader Application NiteTrader that I built. Users are able to create accounts to buy and sell stock, and the app tracks the stock and users performance over time. The app utilizes the Heroku: Cloud Application Platform as the host platform, and implements Heroku Dyno Schedulers to run microservices to update user performances and limit the number of Third-Party API fetches; thus increasing site speed and reliability.
